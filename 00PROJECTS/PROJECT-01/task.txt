REST API -- JSON(JavaScript Object Notation)

GET /users -> Lists all the users -> Done

GET /api/users/1 -> get the user with ID 1  -> Done
GET /api/users/2 -> get the user with ID 2  -> Done

Dynamic PATH Parameters

GET /api/api/users/:id
:id -> variable | dynamic




POST /api/users -> creates new user

PATCH /api/user/id -> edit the user with ID 1 

DELETE /api/users/1 -> Delete the user with ID 1
